# B站视频规范

## 平台特点

- **受众**：年轻用户为主，重视互动
- **形式**：中长视频（10-30 分钟常见）
- **特点**：弹幕文化、社区感强、二次创作活跃
- **内容**：技术、游戏、创作、知识类都很受欢迎

---

## 内容类型

### 1. 技术教程

**核心**：实用、清晰、有干货

**结构**：
```
[开场] 问题引入 + 这期讲什么
[正文] 分步骤演示
[弹幕引导] 关键时刻提醒关注
[结尾] 总结 + 三连引导
```

### 2. 创作分享

**核心**：展示创作过程，有故事性

**结构**：
```
[背景] 为什么做这个
[过程] 创作过程记录
[成果] 最终作品展示
[反思] 经验教训
```

### 3. 知识科普

**核心**：有趣、易懂、有深度

**结构**：
```
[引入] 用故事或问题开场
[展开] 层层递进
[互动] 弹幕提问
[总结] 核心要点
```

## 视频脚本模板

### 技术教程脚本

```markdown
# 标题：[具体问题] + [解决方案]

## 开场（30秒）
「大家好，我是 [名字]」
「今天解决一个很多同学问我的问题：...」
「看完这个视频，你就能...」

## 正文

### 第一部分：[标题]
[内容]
[弹幕引导：「懂的打 1」]

### 第二部分：[标题]
[内容]
[弹幕引导：「有问题的弹幕告诉我」]

### 第三部分：[标题]
[内容]
[常见问题提醒]

## 总结（30秒）
「今天讲了三个要点：...」
「有问题评论区告诉我」

## 结尾
「觉得有用的一键三连」
「下期见」
```

## B站特有规范

### 弹幕引导

在视频中适时引导弹幕互动：

| 时机 | 引导方式 |
|------|---------|
| 讲到痛点 | 「有同感的弹幕扣 1」 |
| 讲到关键点 | 「重点来了，记下来」 |
| 讲到难点 | 「这里不理解的弹幕告诉我」 |
| 结尾 | 「下期想看什么评论区告诉我」 |

### 三连引导

- 视频结尾引导「一键三连」
- 可以制作创意结尾动画
- 口播 + 画面同时引导

### 评论区维护

- 发布后及时回复评论
- 置顶重要信息或补充说明
- 收集下期内容建议

## 标题规范

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 教程 | [问题]？[时间] 教会你 | VSCode 配置太麻烦？5 分钟教会你 |
| 分享 | 我是如何 [达成结果] 的 | 我是如何从零开始学设计的 |
| 科普 | [主题] + [吸引点] | GPT 背后的原理，终于看懂了 |
| 避坑 | 别再 [错误做法] 了 | 别再这样写代码了！三个坏习惯 |

### 封面规范

- 文字简洁有力
- 视觉突出主体
- 风格统一
- 可以加表情/箭头强调

## 与博客的关系

### 博客 → B站

```
博客内容
    │
    → 改写为口语化脚本
    → 添加弹幕引导点
    → 录制视频
    → 简介附博客链接
```

### B站 → 博客

```
视频内容
    │
    → 整理为文字版
    → 添加详细说明
    → 发布博客
    → 视频评论区置顶博客链接
```
