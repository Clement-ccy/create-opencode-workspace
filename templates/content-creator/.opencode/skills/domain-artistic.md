# 艺术/摄影/音乐/电影领域规范

## 概述

艺术领域的写作有其独特的语言和关注点。本文件提供各细分领域的具体规范，作为博客类型规范的补充。

---

## 摄影

### 核心关注点

| 维度 | 描述方式 |
|------|---------|
| 光线 | 硬光/软光、方向、色温、自然光/人工光 |
| 构图 | 三分法、引导线、对称、框架、负空间 |
| 时刻 | 黄金时刻、蓝调时刻、决定性瞬间 |
| 情绪 | 照片传达的感受、氛围 |
| 后期 | 色彩倾向、对比度、质感 |

### 语言特点

- **视觉优先**：用文字重建画面
- **注意光线**：光线是摄影的语言
- **捕捉瞬间**：强调时间感和决定性

### 示例

> 下午四点的光线从窗户斜射进来，在她的侧脸投下一道柔和的轮廓光。我没有调整参数，直接按下快门——那种转瞬即逝的东西，调好了也就消失了。

### 常用词汇

| 类别 | 词汇示例 |
|------|---------|
| 光线 | 侧光、逆光、轮廓光、漫射光、金色时刻 |
| 构图 | 留白、引导线、视觉中心、层次感 |
| 情绪 | 宁静、孤独、温暖、疏离、亲密 |
| 质感 | 颗粒感、柔和、锐利、胶片感 |

## 音乐

### 核心关注点

| 维度 | 描述方式 |
|------|---------|
| 旋律 | 走向、记忆点、重复与变化 |
| 节奏 | 快慢、律动感、切分、停顿 |
| 音色 | 温暖、冷峻、干净、粗糙、电子感 |
| 编曲 | 层次、空间感、乐器搭配 |
| 歌词 | 意象、叙事、情绪 |

### 语言特点

- **节奏感**：句子可以有音乐性
- **通感**：声音可以用视觉语言描述
- **情绪流动**：跟随音乐的情绪变化写作

### 示例

> 前奏很长，几乎有一分钟。钢琴的音色很干，像在空房间里弹。然后人声进来，不是从某个具体位置，而是从四面八方——那种包围感，让人想闭上眼睛。

### 常用词汇

| 类别 | 词汇示例 |
|------|---------|
| 音色 | 温暖、清冷、饱满、单薄、层次分明 |
| 节奏 | 律动、推进、停顿、自由、紧绷 |
| 氛围 | 沉浸、疏离、亲密、宏大、私密 |
| 结构 | 前奏、主歌、副歌、桥段、尾声 |

## 电影

### 核心关注点

| 维度 | 描述方式 |
|------|---------|
| 镜头 | 景别、运动、角度、构图 |
| 剪辑 | 节奏、蒙太奇、转场 |
| 叙事 | 结构、视角、时间线 |
| 表演 | 细节、克制、爆发 |
| 情绪 | 影片传达的整体感受 |

### 语言特点

- **场景感**：重建画面和时刻
- **时间线**：注意叙事的时间感
- **克制评价**：不过度解读，给读者空间

### 示例

> 镜头不动，就停在她脸上。大概有十秒，或者更久——久到你开始注意她的眼眶慢慢变红，注意她嘴角微微的颤抖。没有配乐，没有对白，只是一个长镜头，但我看到了一场完整的崩溃。

### 常用词汇

| 类别 | 词汇示例 |
|------|---------|
| 镜头 | 特写、长镜头、固定镜头、手持、跟拍 |
| 氛围 | 压抑、浪漫、紧张、疏离、温暖 |
| 节奏 | 缓慢、急促、留白、停顿 |
| 表演 | 克制、爆发、细腻、留白 |

## 文学/阅读

### 核心关注点

| 维度 | 描述方式 |
|------|---------|
| 语言 | 风格、节奏、意象 |
| 叙事 | 结构、视角、时间 |
| 人物 | 动机、发展、复杂性 |
| 主题 | 核心问题、隐喻、社会语境 |
| 感受 | 阅读时的情绪体验 |

### 语言特点

- **尊重原文**：引用比转述更有力
- **个人阅读体验**：强调「我读到」而非「客观评价」
- **不过度解读**：保留文本的开放性

### 示例

> 读到这一段的时候，我停了很久。
> 
> 「他不知道自己想要什么，但知道自己不想要什么。」
> 
> 这句话写得那么平静，但好像什么都说透了。

## 跨领域通用规范

### 描述 vs 评价

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| 纯描述 | 「这张照片用了逆光」 | 技法探讨 |
| 描述+感受 | 「逆光让她的轮廓有一种朦胧感，像是记忆」 | 作品赏析 |
| 纯评价 | 「这张照片拍得很好」 | ⚠️ 尽量避免 |

### 「我」的使用

在艺术写作中，**「我」是合法的**：

- 「我看到...」
- 「这让我想起...」
- 「我不知道为什么，但...」

个人视角不是偏见，是艺术写作的核心。

### 避免的写法

| 避免 | 原因 |
|------|------|
| 「这本作品深刻地揭示了...」 | 太像论文，太空洞 |
| 「伟大的杰作」 | 空洞的评价 |
| 「建议所有X爱好者都...」 | 过度推荐 |
| 「客观来说...」 | 艺术没有客观 |
