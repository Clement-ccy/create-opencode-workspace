# TikTok/抖音/Reels 短视频规范

## 平台特点

- **受众**：广泛，注意力短暂
- **形式**：短视频（15-60 秒）
- **特点**：强算法推荐、视觉优先、快节奏
- **语言**：根据平台选择（抖音中文，TikTok 英文/多语言）

---

## 内容类型

### 1. 技巧片段

**核心**：一个技巧一个视频

**结构**：
```
[0-3秒] 问题引入/钩子
[3-45秒] 技巧演示
[45-60秒] 总结/行动呼吁
```

### 2. 精华总结

**核心**：把长内容的精华浓缩

**结构**：
```
[钩子] 「这三个技巧改变了我...」
[要点1] 快速展示
[要点2] 快速展示
[要点3] 快速展示
[结尾] 「完整版在主页/置顶」
```

### 3. 快速教程

**核心**：极简的教学

**结构**：
```
[开场] 你将学会什么
[演示] 一步步快速展示
[结尾] 鼓励尝试/关注
```

## 短视频脚本模板

### 技巧类

```markdown
时长：15-30秒

[开场 - 3秒]
「这个技巧我用了三年」
[画面：问题场景]

[演示 - 20秒]
「只需要三步...」
[画面：快速演示]
[字幕：步骤1/2/3]

[结尾 - 5秒]
「试试看」
[画面：结果展示]
[字幕：关注获取更多]
```

### 精华总结类

```markdown
时长：30-60秒

[开场 - 3秒]
「今天学了三个改变认知的观点」
[画面：有吸引力的开场]

[要点1 - 15秒]
「第一，...」
[画面：配合说明]
[字幕：核心观点]

[要点2 - 15秒]
「第二，...」
[画面：配合说明]

[要点3 - 15秒]
「第三，...」
[画面：配合说明]

[结尾 - 5秒]
「完整版看我主页」
[字幕：关注我]
```

## 制作要点

### ✅ 应该做的

| 要点 | 说明 |
|------|------|
| 前 3 秒必须抓住 | 决定是否划走 |
| 字幕必不可少 | 很多人静音看 |
| 节奏要快 | 避免冗长 |
| 单一主题 | 一个视频一个点 |
| 视觉变化 | 画面不要太单一 |

### ❌ 避免的

| 避免 | 原因 |
|------|------|
| 开场拖沓 | 观众会划走 |
| 内容太多 | 记不住 |
| 无字幕 | 损失大量观众 |
| 画面单调 | 留存率低 |
| 只有口播无画面 | 枯燥 |

## 系列内容

短视频适合做系列：

### 系列规划

```
主题：[大主题]

第1期：[子主题A]
第2期：[子主题B]
第3期：[子主题C]
...

每期结尾：「下期讲 [预告]，关注不错过」
```

### 系列优势

- 算法会推荐系列其他视频
- 观众有追更动力
- 个人风格更统一

## 与博客的关系

### 博客 → 短视频

```
博客内容
    │
    → 拆分为多个小点
    → 每个点做成一个短视频
    → 形成系列
    → 主页/简介附博客链接
```

**示例**：
```
一篇博客：「5 个提高效率的方法」
    │
    → 视频1：「方法1 - [具体技巧]」
    → 视频2：「方法2 - [具体技巧]」
    → 视频3：「方法3 - [具体技巧]」
    → ...
    → 汇总视频：「5 个方法总结」
```

### 引流策略

- 简介放博客/公众号链接
- 置顶视频引导关注
- 评论区分发完整内容链接
