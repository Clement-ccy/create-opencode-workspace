# YouTube 视频规范

## 平台特点

- **受众**：全球用户，偏好深度内容
- **形式**：中长视频为主（5-15 分钟）
- **特点**：SEO 友好，长尾流量，可变现
- **语言**：根据目标受众选择语言

---

## 内容类型

### 1. 教程视频

**核心**：手把手教会观众

**结构**：
```
[0:00-0:30] 开场 + 本期内容预告
[0:30-1:00] 背景说明 + 你将学到什么
[1:00-主体] 分步骤演示
[结尾] 总结 + 下期预告 + 订阅引导
```

### 2. 深度分析

**核心**：深入探讨一个话题

**结构**：
```
[开场] 引入问题
[背景] 为什么重要
[分析] 层层展开
[结论] 核心观点
[互动] 评论区讨论
```

### 3. 工具/产品介绍

**核心**：展示工具的价值

**结构**：
```
[开场] 这个工具解决什么问题
[演示] 实际使用展示
[对比] 与其他方案对比
[建议] 适合谁/不适合谁
[链接] 描述栏附链接
```

## 视频脚本模板

### 从博客到视频脚本

```markdown
# 视频标题：[具体且有吸引力的标题]

## 开场（30秒）
[场景/问题引入]
「你有没有遇到过这种情况...」
「今天我们来聊...」

## 内容预告（15秒）
「看完这个视频，你将学会...」

## 主体内容

### 第一部分：[标题]
[内容]
[画面说明：屏幕录制/示意图/实拍]

### 第二部分：[标题]
[内容]
[画面说明]

### 第三部分：[标题]
[内容]
[画面说明]

## 总结（30秒）
[回顾要点]
「今天我们讲了三个要点：1... 2... 3...」

## 结尾（15秒）
「如果这个视频对你有帮助，请点赞订阅」
「评论区告诉我你的想法」
```

## 制作要点

### ✅ 应该做的

| 要点 | 说明 |
|------|------|
| 前 30 秒抓住注意 | 决定观众是否继续看 |
| 清晰的章节划分 | 便于跳转和搜索 |
| 添加字幕 | 提升可访问性 |
| 描述栏信息完整 | 链接、时间戳、简介 |
| 封面图有吸引力 | 决定点击率 |

### ❌ 避免的

| 避免 | 原因 |
|------|------|
| 开场太长 | 观众会离开 |
| 无章节标记 | 体验差 |
| 无字幕 | 限制受众 |
| 封面图无信息 | 点击率低 |
| 标题太模糊 | 搜索难发现 |

### 标题规范

| 类型 | 公式 | 示例 |

## 与博客的关系

### 博客 → YouTube

```
博客内容
    │
    → 改写为口语化脚本
    → 添加视觉元素说明
    → 录制视频
    → 描述栏附博客链接
```

### YouTube → 博客

```
视频内容
    │
    → 整理为文字稿
    → 添加细节和扩展
    → 发布博客
```
