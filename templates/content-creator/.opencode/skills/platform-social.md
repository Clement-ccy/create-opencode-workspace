# 社交媒体图文平台分发规范

此技能用于将博客文章转化为适合社交媒体图文平台（短平快阅读）发布的内容。

## 适用平台
- 高视觉型：小红书 (Xiaohongshu)、Instagram
- 资讯/观点型：Twitter (X)、微博 (Weibo)、即刻 (Jike)

## 转化原则

1. **吸睛标题**：前三行决定生死。使用具有悬念、痛点或价值承诺的标题。
2. **信息压缩**：不要复制整段文章。提取最核心的 1-3 个观点。如果文章较长，考虑将其拆分为一个“系列贴”或“Thread”。
3. **视觉排版**：充分利用 Emoji 进行视觉引导和段落分割。多留白，绝对不要出现连续超过 5 行的文字块。
4. **强行动呼吁 (CTA)**：结尾必须有明确的互动引导（提问、求建议、引导收藏）。

## 平台特定微调（如果用户未指定，请遵循以下默认调整）

### 针对 小红书 / Instagram (高视觉型)
- 首图文案必须具有冲击力。
- 文本重点在于提供**具体的解决方案**或**情绪共鸣**。
- 正文排版需要大量的换行和清晰的要点提取。
- 结尾加上大量相关 Hashtags（至少 5-8 个）。

### 针对 Twitter / 微博 / 即刻 (资讯/观点型)
- 控制字数（Twitter 如果不是长推文，请分成 Thread 形式，每条 280 字符以内）。
- 语气更加锐利、直接，抛出观点或反常识结论。
- Hashtags 精简（1-3 个）。

## 输出格式要求

```markdown
【标题】（20字以内，包含关键词和情绪）

【引言】（直接抛出痛点或结论）

【核心内容 1】
[Emoji] ...

【核心内容 2】
[Emoji] ...

【结论/建议】
...

【互动引导/CTA】
...

【Tags】
#标签1 #标签2 #标签3
```

## 注意事项

- 不要使用“总结一下”、“总而言之”这种僵硬的过渡词。
- **降维表达**：确保非专业读者也能立刻看懂。将专业术语转化为日常比喻。
- 如果文章本身属于 `domain-personal`（个人随笔），社交媒体转化应该更侧重于“树洞吐槽”或“真实情感抒发”，而不是“干货教学”。
