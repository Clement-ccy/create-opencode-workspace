# 数据分析与可视化

## 概述
本技能用于实验结果分析、可视化与表格展示。适用于 `/analyze`。

## 分析工作流
```text
收集 → 清洗 → 分析 → 可视化 → 解释
```

## 统计方法
- **描述统计**：均值、方差、分位数。
- **假设检验**：t-test / Wilcoxon / bootstrap。
- **置信区间**：95% CI，报告方法。
- **效应量**：Cohen's d 等。

## 可视化规范
| 图表类型 | 适用场景 | 注意事项 |
|---|---|---|
| 条形图 | 方法对比 | 显示误差条 |
| 折线图 | 趋势 | 统一坐标尺度 |
| 散点图 | 相关性 | 标注相关系数 |

### 颜色与可读性
- 使用色盲友好配色。
- 不使用纯红绿对比。

### 坐标轴规范
- 标注单位与尺度。
- 避免截断导致误导。

### 图注写作模板
```text
图 X：展示 [方法/变量] 在 [数据集] 上的 [指标]，误差条为标准差。
```

## 结果表格规范
- 最优结果 **加粗**。
- 必须包含标准差。
- 标注显著性标记（*, **）。

## analytical_plots/ 描述格式
```text
文件名: <plot-name>.png
目的: <比较/趋势/相关性>
数据来源: <实验名>
关键结论: <一句话>
```

## 常见陷阱
- 选择性展示结果（cherry-picking）。
- 隐藏误差或不报告方差。
- 使用误导性坐标轴。

## 检查清单
- [ ] 数据清洗步骤记录完整
- [ ] 图表含误差条
- [ ] 表格含标准差与显著性
- [ ] 坐标轴与单位清晰
- [ ] 结论与图表一致
