# 研究提案写作语调

## 概述
本技能用于研究提案与想法陈述。目标是清晰、具有说服力但保持可行性。

## 写作风格
- **前瞻性**：强调未来工作与潜在影响。
- **问题驱动**：清楚说明问题的重要性。
- **可行性**：提出实验或方案可落地。

## 结构要点
### 问题陈述
- 说明现有方法局限。
- 给出可验证研究问题。

### 新颖性
- 明确“新在哪”：方法/应用/分析。
- 与相关工作对比，避免泛泛而谈。

### 可行性评估
- 数据可获得性。
- 计算资源与时间成本。
- 预期风险与缓解策略。

## 与论文写作区别
| 维度 | 提案 | 论文 |
|---|---|---|
| 视角 | 前瞻/计划 | 回顾/总结 |
| 证据 | 假设与预期 | 实验与结果 |
| 语气 | 说服力 | 证据驱动 |

## 常见错误
- 过于宏大，缺乏可行性。
- 新颖性描述空洞。
- 未说明与已有工作差异。

## 研究提案模板
```text
背景与动机
→ 研究问题
→ 现有方法不足
→ 本文想法与新颖性
→ 可行性与计划
→ 风险与缓解
```

## 检查清单
- [ ] 研究问题明确且可验证
- [ ] 新颖性与相关工作对比清晰
- [ ] 可行性分析具体
- [ ] 风险与缓解措施明确
